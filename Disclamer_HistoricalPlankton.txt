3. DATA SPECIFIC INFORMATION
The Historical Plankton website (https://ecosystemdata.ices.dk/HistoricalPlankton/Download.aspx) provides the user with the ability to query the Historical Plankton database and download the data records.
Download ZIP bundle includes the requested data file in the CSV format, Historical Plankton fact sheet, and the data disclaimer.  
Data are limited to years 1901-1912 and are based on the publications:
 - ICES Bulletin Resultats Acquis Pendant les Courses Periodiques 1901-1902
 - ICES Bulletin Resultats Acquis Pendant les Courses Periodiques 1902-1903
 - Resultats Acquis Pendant les Courses Periodiques 1903-1904
 - Resultats Acquis Pendant les Courses Periodiques 1904-1905
 - Resultats Acquis Pendant les Courses Periodiques 1905-1906
 - Resultats Acquis Pendant les Courses Periodiques 1906-1907
 - Bulletin Planktonique 1907-1912
Note that additional data, such as methods and quality assurance may be available. To have access to the complete data set, the user can send a request to accessions@ices.dk, specifying the download code and the extra information required.
